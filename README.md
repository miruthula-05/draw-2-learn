# Draw to Learn 🎨📚

## Overview

Draw to Learn is an AI-powered educational storytelling system that converts children's drawings into narrated learning videos. The system uses computer vision and natural language processing to interpret drawings and generate simple educational stories that help children learn through creativity.

This project focuses on making learning more interactive by transforming children's imagination into engaging educational content.

---

## Problem Statement

Children often express ideas through drawings, but traditional educational platforms do not utilize these drawings as a learning tool. There is a gap between creativity and structured learning experiences.

---

## Solution

Draw to Learn bridges this gap by using AI to analyze children's drawings and convert them into narrated educational stories. The system recognizes elements in the drawing and generates simple stories that explain concepts in an engaging way.

---

## Features

* Detects objects from children's drawings using computer vision
* Generates educational stories based on detected objects
* Converts text stories into narrated audio
* Creates an engaging learning experience through AI storytelling

---

## Tech Stack

* Python
* OpenCV
* NLP
* Generative AI
* NLTK

---

## System Workflow

1. User uploads a drawing.
2. Computer Vision detects objects in the drawing.
3. NLP generates a simple educational story.
4. The story is converted into narrated output.
